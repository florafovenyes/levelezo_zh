# II. Zárthelyi dolgozat – Gyakorlati rész

## Általános tudnivalók

A repository-ban található feladatokat kell megoldani a rendelkezésre álló idő alatt.

A megoldás során:

- használható Python dokumentáció
- használható internetes keresés (pl. StackOverflow)
- használható AI eszköz (pl. ChatGPT, Copilot)
- **NEM ENGEDÉLYEZETT** bármilyen kommunikáció más hallgatókkal vagy külső személyekkel

## Fontos szabályok

A beadott megoldás minden sorát érteni kell.

A javítás során figyelembe vehető:

- működőképesség
- helyes logika
- olvasható kód
- hibakezelés
- részleges megoldás is ér pontot

## Feladattípusok

Az alábbi feladattípusok szerepelnek a dolgozatban:

1. Szintaktikai hibás kód javítása  
2. Logikai / szemantikai hibás kód javítása  
3. Meglévő program kiegészítése új funkcióval  
4. Új programrész implementálása
5. UML diagram generálása feladat alapján

## Munkamenet

1. Forkold ezt a repository-t saját GitHub accountodra.
2. Clone-old és nyisd meg PyCharm-ban a tanult módon.
3. Oldd meg a feladatokat a /feladatok mappában.
4. Commit & Pushold a kész megoldásokat a tanult módon.

## Beadás

A ZH végén a saját forkolt repository linkjét kell elküldeni emailben
az oktató emailcímére.
A tárgyban szerepeljen a "ZH II. - [Saját neved]" szöveg.

## AI használat dokumentálása

A nem dokumentált AI használatot 100%-ban saját munkának fogom tekinteni.
**Az ilyen megoldásokba a kötelező program bemutatása során bele fogok kérdezni.**

Az **AI_REFERENCES.md** fájlba kerüljön dokumentálásra minden AI eszköz használata, mely a megoldás során alkalmazásra került. 
Minden egyes AI eszköz használatát külön szekcióban kell dokumentálni, mely tartalmazza:

- milyen AI eszközt használtál
- mire használtad
- beszélgetés linkje(i)
